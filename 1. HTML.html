1. Porque usar HTML?

    ¿Que es HTML?

    HTML son las siglas de Hypertext Markup Lenguage, que viene a significar, lenguaje de marcación de hipertexto.

    De forma resumida, HTML no es mas que un lenguaje usado para crear paginas web por medio de marcadores y atributos, que definen la estrucutra que va a tener la pagina web

    Elementos principales

    Pero primero es necesario recordar que estos tags son identificadas con un signo de menor que (<), seguido del nombre de la etiqueta y un signo mayor que (>). La mayoría de los tags deben ser cerradas.

    Ejemplos de como funcionaria:

      > Parrafo (<p>)

      El tag <p>; debe abrirse y cerrarse para definir un párrafo.

      <p>Este seria un párrafo en HTML.</p>

      > Encabezado (<h1>, <h2>, <h3>)

      Las heading tags o las etiquetas de cabecera, ayudan a crear una jerarquía entre las partes del texto, que lo separa en secciones. El <h1> es el encabezado más relevante, seguido del <h2>, <h3> y así sucesivamente al <h6>, de menor importancia.

      <h1>Título principal</h1>

      <h2>1. Primer intertítulo</h2>

      <h3>1. Primer intertítulo</h3>

      > Imagen (<img>)

      No es necesario cerrar el tag de la imagen, ya que su contenido se define en la etiqueta con el atributo «src».

      <img src=»https://acortar.link/c395p1»>

    Estos serian uno ejemplos muy basico para poder crear una pequeña estructura de html. Como se ha podido observa es muy simple usar html lo que facilita la creación de nuevas paginas web.
